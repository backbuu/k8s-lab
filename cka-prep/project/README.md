# Project — Capstone: Pass the CKA

The capstone is not an app — it is **repeatedly passing full-length, timed CKA mock exams**.

## Definition of done
- ≥ **66%** on a full 2-hour mock, achieved on **2 consecutive** mocks.

## How to run a mock
1. Spin up a fresh multi-node cluster (kind/kubeadm). Break nothing in advance.
2. Set a **2-hour** timer. Open only kubernetes.io docs.
3. Work a full task set (Killer.sh session, killercoda set, or a self-authored set drawing from all 5 domains).
4. Stop at time. Grade honestly. Record below and in `../progress.md`.
